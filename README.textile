Major League Baseball Score for MetaWatch Community Edition

This is the source of an Android app which works in conjunction with the MetaWatch Community Edition
to display information about a Major League Baseball game on a MetaWatch. This program is distributed
under the Apache License Version 2.0. See the LICENSE file.

The latest binary download for MetaWatch Community Edition can be found on the "downloads" page on github:
https://github.com/benjymous/MWM-for-Android/downloads/

and on the Google Play Store:
https://play.google.com/store/apps/details?id=org.metawatch.communityedition

For details about this program read the file assets/about.html.
